Circle Rules

Things a user can do with his or her circles are:

  • Create a circle
  • Search for a user and add him to a circle
  • Make a post
  • Comment on a post
  • Like a post
  • Like a comment
  • Remove a user from a circle
  • Remove a post
  • Remove a comment
  • Unlike a post
  • Unlike a comment
  • Modify a post
  • Modify a comment
  • Delete a circle
  • Rename a circle
  • Join a circle
  • Unjoin a circle
